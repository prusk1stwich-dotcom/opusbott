# 🤖 Materials for admins of OpusGuru — Telegram Bot

## Установка и запуск

### 1. Требования
- Python 3.10 или выше

### 2. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 3. Запуск
```bash
python bot.py
```

---

## Как работает бот

### Доступ
Бот доступен только для указанных пользователей по их Telegram ID:
- `8531243207`, `8422967271`, `791428991`, `7201046344`, `6359224120`, `599952947`

Все остальные получат сообщение об отсутствии доступа.

---

### Обычные пользователи видят 2 кнопки:
| Кнопка | Действие |
|--------|----------|
| 📢 Актуальное сообщение для рассылки | Показывает текущее сообщение для рассылки |
| 📂 Папки и чаты для рассылок | Показывает список папок и чатов |

---

### Администраторы (ID: 599952947, 8531243207, 7201046344) видят 4 кнопки:
| Кнопка | Действие |
|--------|----------|
| 📢 Актуальное сообщение для рассылки | Показывает текущее сообщение |
| 📂 Папки и чаты для рассылок | Показывает список папок и чатов |
| ✏️ Обновить сообщение для рассылки | Обновить текст сообщения для рассылки |
| 🗂 Обновить папки и чаты | Обновить список папок и чатов |

---

### Как обновить материалы (для администраторов):
1. Нажмите кнопку **✏️ Обновить сообщение для рассылки** или **🗂 Обновить папки и чаты**
2. Отправьте новый текст в чат
3. Бот сохранит изменения — они сразу станут доступны всем пользователям
4. Чтобы отменить действие — напишите `/cancel`

---

### Хранение данных
Все материалы хранятся в файле `materials.json` рядом с ботом.  
Файл создаётся автоматически при первом обновлении материалов.
